# ♥️ ♣️ BLACK JACK ♦️ ♠️

A Multiplayer Black Jack game. Join rooms, play with other players and enjoy!

## Technologies

- Typescript
- Vite
- Node.js
- Express
- Websockets
- Tailwind

## Features

- [x] Core Black Jack game logic
- [x] Join or create room by URL
- [ ] Screen for create or join room
- [ ] Card "A" value 1 or 10
- [ ] Show spectators
- [ ] Room Chat
- [ ] Player bets
